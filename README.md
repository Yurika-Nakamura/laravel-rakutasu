# booby

楽にタスクを管理するためのアプリです。

# Getting Started

```bash
cd プロジェクトルート
cp .env.example .env
composer install
./vendor/bin/sail up -d
./vendor/bin/sail artisan migrate --seed
./vendor/bin/sail artisan serve
```

[http://localhost:8000](http://localhost:8000) ポートが立ち上がっていることが確認できれば OK です。
